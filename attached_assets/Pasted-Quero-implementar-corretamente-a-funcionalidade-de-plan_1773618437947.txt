==================================================
REGRA DE NEGÓCIO PRINCIPAL
==================================================

No domínio protection_network, o sistema deve permitir cadastrar:

Endereço
  -> Planta
      -> Medidas

Exemplos de plantas:
- Sacada reta
- Sacada em L
- Apto 63 m²
- Apto 86 m²
- Apto 115 m²

A planta é uma identificação do tipo/configuração do imóvel.
Ela NÃO é cálculo automático.
Ela deve ser cadastrada manualmente.

==================================================
REGRA DO BOT
==================================================

Quando o cliente enviar um endereço:

1. O sistema deve buscar o endereço no catálogo
2. Se não existir:
   - seguir fluxo normal de coleta manual de medidas

3. Se existir:
   - verificar quantas plantas existem para o endereço

CASO A — só existe 1 planta
- usar automaticamente essa planta
- seguir o fluxo do orçamento

CASO B — existem 2 ou mais plantas
- o bot deve perguntar ao cliente qual é a planta

IMPORTANTE:
O bot deve SEMPRE mostrar o nome da planta ao cliente.
O nome da planta NÃO faz parte da regra de ocultação de medidas.

Exemplo correto:

"Encontrei mais de uma planta para esse endereço.

Qual é a sua planta?"

1) Sacada reta
2) Sacada em L
3) Apto 63 m²
4) Apto 86 m²
5) Apto 115 m²

Depois que o cliente escolher:
- carregar apenas as medidas da planta escolhida
- seguir o fluxo normal

==================================================
REGRA DO TOGGLE
==================================================

Existe uma configuração:
company_settings.extra_settings.protection_network.show_measures_to_customer

Essa configuração controla apenas a exibição das medidas ao cliente.

Se show_measures_to_customer = true:
- o bot pode mostrar medidas
- pode listar áreas
- pode mostrar nomes de áreas
- pode mostrar descrições
- pode deixar o cliente escolher áreas

Se show_measures_to_customer = false:
- o bot NÃO pode mostrar medidas
- NÃO pode mostrar lista de áreas
- NÃO pode mostrar nomes de medidas
- NÃO pode mostrar descrição de medidas
- NÃO pode mostrar largura/altura
- NÃO pode mostrar quantidade de áreas

MAS, MESMO COM O TOGGLE DESLIGADO:
- o bot DEVE continuar mostrando o nome da planta
- o bot DEVE continuar perguntando qual planta o cliente possui
- o bot DEVE continuar usando as medidas internamente para cálculo do orçamento

Resumo:
- planta = sempre pode aparecer
- medidas = ocultar ou mostrar conforme toggle
