PARTE 4 — TEMA CLARO / ESCURO PERSISTENTE
==================================================

Quero adicionar alternância de tema:
- claro
- escuro

Regras:
- cada usuário escolhe o tema
- a escolha deve ser salva permanentemente
- ao fazer login novamente, o tema deve voltar como estava
- isso deve funcionar para qualquer usuário:
  - master_admin
  - company_admin
  - seller
  - installer

Se possível:
- salvar a preferência no backend por usuário
- se não houver estrutura pronta, usar o melhor caminho escalável

Adicionar:
- botão/toggle de tema na interface
- persistência confiável
